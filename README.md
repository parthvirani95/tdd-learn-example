### You can use it, if you want to learn the following topics:

- Clean Architecture based on Reso Coder Approach
- DI (Depedency Injection)
- Unit Testing
- Widget Testing
- Bloc Testing
- Mockito
- Json serialization
- Bloc State Management
- Multiple Result Management
- Shared Preferences
- Internet Connectivity
- Retrofit
